What is Metric Drift?

Metric drift occurs when a store's KPI deviates more than 12% from the cross-store baseline. The system queries the Neptune business graph for linked nodes — only the relevant context for each metric — then generates root-cause analysis and recommendations.

Graph Traversal Per Metric

Gross Margin→ Product → Category → Brand → Supplier
Return Rate→ Product → Customer → LoyaltyTier
Avg Order Value→ Customer → Loyalty → Promotion
Discount Rate→ Promotion → Type → Channel
Net Sales→ Category → Brand → Region

Loss Projection Formula

Daily Sales = Net Sales / 30
Impact Rate = |Drift %| / 100
Loss = Daily × Impact × Multiplier × Days
Multipliers: Return Rate 1.2× · Gross Margin 0.8× · Discount 0.9× · AOV 0.6× · Net Sales 1.0×
